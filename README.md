# StiSupportForm

Fill in the STI support form which will be game agnostic.

## Installation

Add this line to your application's Gemfile:

    gem 'sti_support_form'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sti_support_form

## Usage

Fill in the STI support form which will be game agnostic.
